Follow these simple steps to launch your password manager:

2.  **Open Command Prompt:**

    In the black command window that appears, type the following command and press **Enter**:

    ```bash

    ```

    **OR** simply double-click **`Namo.exe`** inside the `CoralReefs` folder!
